Δείτε 7 συμβουλές για το πώς να αγοράσετε γυαλιά οράσεως online.

Στις μέρες μας τα ¾ του πληθυσμού χρειάζονται κάποιου είδους διόρθωση στην όρασή τους. Αυτό μας έδωσε κίνητρο να κάνουμε την online αγορά γυαλιών οράσεως πιο εύκολη για το χρήστη. Διαβάσετε παρακάτω 7 συμβουλές για τη σωστή επιλογή σκελετού και φακών. 

1. Το πιο σημαντικό για την κατασκευή ενός σωστού σκελετού οράσεως είναι να έχετε μια πρόσφατη συνταγή από το γιατρό σας.

Πολλοί χρήστες έρχονται σε εμάς για να κατασκευάσουν τα γυαλιά τους οnline. Η πρώτη μας ερώτηση είναι, από πότε είναι η συνταγή σας. Είναι κρίμα να κατασκευάσετε ένα καινούργιο ζευγάρι και να έχουν αλλάξει τα νούμερά σας.
Η τυπική ισχύς μιας συνταγής είναι 2 έτη για τους ενήλικες και 1 έτος εάν είστε νεότεροι. Όταν εξετάζετε την όρασή σας, ο οπτομέτρης ή ο οφθαλμίατρος πρέπει να σας δώσει ένα αντίγραφο της συνταγής.

Εδώ είναι μερικές συντομογραφίες και όροι που αναφέρονται στη συνταγή σας θα πρέπει να γνωρίζετε όταν παραγγέλνετε γυαλιά online:

Το ΔΟ (Δεξιός οφθαλμός) αναφέρεται στο δεξί σας μάτι.
Το ΑΟ (Αριστερός οφθαλμός) αναφέρεται στο αριστερό σας μάτι.
Η σφαίρα (SPH) υποδεικνύει τη μυωπία (-) ή της υπερμετρωπία σας (+).
Το ADD (Addition) αφορά τη κοντινή σας όραση (πρεσβυωπία).
Ο κύλινδρος (CYL) υποδεικνύει τον αστιγματισμό.
Ο άξονας περιγράφει τη θέση της κυλινδρικής ισχύος στο μάτι σας (απαιτείται για αστιγματισμό).

Σημείωση: αν θέλετε να χρησιμοποιήσετε τη συνταγή σας για αγορά φακών επαφής, είναι πιθανόν να χρειαστεί μετατροπή, για αυτό ρωτήστε τον οπτικό σας.

2. Πάνω στη συνταγή σας θα πρέπει να αναγράφετε η διακορική απόσταση (PD)

Αυτή η μέτρηση είναι εξίσου σημαντική με την ίδια τη συνταγή. Χωρίς τη διακορική απόσταση δε θα μπορέσουν να κατασκευαστούν σωστά τα γυαλιά σας. Πολλοί χρήστες μας ρωτάνε αν θα μπορούσαν να το κάνουν μόνοι τους. Πάντα προτρέπουμε η μέτρηση αυτή να γίνετε από κάποιον ειδικό.

Η απόσταση αυτή κυμαίνεται γενικά μεταξύ 54mm και 68mm.

Για το λόγο αυτό δεν προτείνουμε ποτέ τα έτοιμα γυαλιά για διάβασμα. Τα γυαλιά αυτά σπάνια θα ταιριάξουν με τη διακορική σας απόσταση με αποτέλεσμα η όρασή σας να μην είναι η σωστή.

Σημείωση: Ο οφθαλμίατρος κατά τη διάρκεια της οφθαλμολογικής εξέτασης ενδεχομένος να παραλείψει να την κάνει ή να τη γράψει στη συνταγή σας. Απλά να του το αναφέρετε για να το σημειώσει.

3. Να είστε προσεκτικοί στην επιλογή του σωστού σκελετού

Η αλήθεια είναι ότι η αγορά γυαλιών οnline είναι πιο δύσκολη από άλλα είδη όπως πχ παπούτσια, υπολογιστές κ.α., για αυτό θα πρέπει να ακολουθείτε κάποια tips.

- Επιλέξτε γυαλιά που συμπληρώνουν το σχήμα του προσώπου σας

Ένας σωστός σκελετός οράσεως δίνει (πέρα από την καλύτερη όραση) τη δυνατότητα να συμπληρώνει τα χαρακτηριστικά του προσώπου σας και τη συνολική σας εμφάνιση. Δημιουργήστε μια ισορροπημένη εμφάνιση χρησιμοποιώντας γυαλιά επιλέγοντας ένα σχήμα σκελετού που παρέχει μια αντίθεση με το σχήμα του προσώπου σας. Δείτε τον παρακάτω οδηγό.

- Επιλέξτε ένα πλαίσιο που ταιριάζει στο προσωπικό σας στυλ

Ποιες είναι οι προσωπικές και επαγγελματικές σας ανάγκες;

- Η δουλειά σας επιβάλει κουστούμι: Δείτε πρώτα τα συντηρητικά χρώματα και σχήματα πλαισίου. Για να βελτιώσετε την επαγγελματική σας εικόνα, σκεφτείτε κλασικά σχήματα όπως οβάλ και ορθογώνια. Συνιστώνται μαύρα, γυαλιστερά, ασημένια και καφέ πλαίσια, επειδή αυτά τα συντηρητικά χρώματα ταιριάζουν με τα επαγγελματικά κοστούμια.
- Για δημιουργικούς επαγγελματίες: Φορέστε μοντέρνα σχήματα, όπως γεωμετρικά σχέδια με χοντρά ή μεταλλικά πλαίσια. Δοκιμάστε ασυνήθιστα χρώματα - μπλε, μοβ και πράσινο. Τα ρετρό και τα vintage είναι επίσης κατάλληλα για να αναδείξουν τη δημιουργική άκρη σας.
- Για τους μαθητές: Ασυνήθιστα σχήματα, φωτεινά χρώματα, μεγαλύτερα μεγέθη και ενδιαφέρουσες λεπτομέρειες, όπως χρωματιστά ελάσματα. Νιώστε ελεύθεροι να πειραματιστείτε.
- Για τον πολυάσχολο γονέα: Μην ανησυχείτε για τις τελευταίες τάσεις στα γυαλιά, παραμείνετε σε ένα βασικό αλλά κομψό ζευγάρι γυαλιών. Οι οβάλ, τα ανάγλυφα ορθογώνια και τα μαλακά σχήματα γάτας είναι πολύ λειτουργικά και εξακολουθούν να φαίνονται υπέροχα. Τα πιο σκούρα χρώματα, όπως το βαθύ κόκκινο, το μαύρο και το πράσινο, μπορούν επίσης να προσθέσουν στο στυλ σας.

- Αγοράστε το σωστό μέγεθος

Η μόδα είναι δευτερεύουσα. Η σωστή τοποθέτηση των φακών μπροστά στα μάτια σας πρέπει να είναι το κύριο μέλημά σας.

Εάν έχετε ήδη ένα ζευγάρι γυαλιών που σας ταιριάζουν καλά, ελέγξτε τις μετρήσεις που είναι γραμμένες στον εσωτερικό βραχίονα του πλαισίου. Σε ορισμένες περιπτώσεις, οι αριθμοί θα εκτυπωθούν πάνω ή ακριβώς πίσω από τη γέφυρα της μύτης. Οι αριθμοί αντιπροσωπεύουν τις ακόλουθες μετρήσεις:

Μέγεθος ματιών - το οριζόντιο πλάτος του φακού, ένας διψήφιος αριθμός στην περιοχή των 40 - 62mm.
Μέγεθος γέφυρας - η οριζόντια απόσταση μεταξύ των δύο φακών, ένας διψήφιος αριθμός στην περιοχή 14 - 24mm.
Μέγεθος βραχίωνα - ένας τριψήφιος αριθμός στην περιοχή των 120 - 165mm.

4. Αγοράστε το σωστό σκελετό

Υπάρχουν δύο επιλογές για την αγορά γυαλιών οράσεως - στο τοπικό οπτικό κατάστημα ή μέσω ηλεκτρονικού καταστήματος.

Πλεονεκτήματα αγοράς offline:

Επαγγελματική βοήθεια - Ένας έμπειρος οπτικός έχει τις γνώσεις για να σας πει εάν ο σκελετός που επιλέγετε είναι κατάλληλος για τη συνταγή σας ή εάν ταιριάζει σωστά στη μύτη σας.
Σύνθετες συνταγές - Εάν έχετε υψηλούς βαθμούς ή περίπλοκη συνταγή, είναι δύσκολο να χρησιμοποιήσετε ένα ηλεκτρονικό κατάστημα. Ένας έμπειρος οπτικός είναι απαραίτητος για να σας βοηθήσει να αποκτήσετε τα σωστά γυαλιά.
Υποστήριξη τοπικών - Εάν θέλετε να υποστηρίξετε τα τοπικά καταστήματά σας, συγκρίνεται την τιμή του ίδιου σκελετού γυαλιών online.

Πλεονεκτήματα αγοράς online:

Χαμηλότερο κόστος - Σε πολλές περιπτώσεις, μπορείτε να εξοικονομήσετε έως και 50%.
Περισσότερες επιλογές - Τα ηλεκτρονικά καταστήματα έχουν συνήθως μια ευρύτερη γκάμα γυαλιών. Οι φακοί που προσφέρονται είναι ίδιας ποιότητας με εκείνους που διατίθενται στα καταστήματα λιανικής πώλησης. Τα γυαλιά κατασκευάζονται από επαγγελματίες οπτικούς σε εργαστήρια.
Ευκολία και ταχύτητα - δεν χρειάζεται να χάσετε χρόνο πηγαίνοντας σε ένα κατάστημα. Παραγγείλετε από την ιδιωτικότητα του σπιτιού σας και τα γυαλιά σας θα παραδοθούν στην πόρτα σας.

5. Κατανοήστε τις ανάγκες σας και προσθέστε φίλτρα

Πλέον η τεχνολογία των φακών και των φίλτρων έχουν εξελιχθεί. Πλεόν ο οπτικός σας μπορεί να προτείνει το φακό που ταιριάζει στη δική σας ανάγκη. Για παράδειγμα αν η δουλειά σας απαιτεί πολύωρη χρήση υπολογιστή, αν δουλεύετε σε εξωτικό χώρο ή αν είστε οδηγός. 

Αν δεν είστε σίγουρος μπορείτε να καλέσετε και κάποιος εξειδικευμένος συνεργάτης μας θα σας ενημερώσει για όλα όσα πρέπει να ξέρετε.

6. Εξετάστε την αγορά περισσότερων από ένα ζευγάρι γυαλιά

Όταν αγοράζετε γυαλιά οράσεως από έναν παραδοσιακό έμπορο λιανικής πώλησης, το κόστος περιλαμβάνει κάποιες πρόσθετες αξίες που ανεβάζουν το τελικό κόστος. Όταν υπολογίζετε το κόστος κατασκευής, το κόστος του διανομέα, το κόστος αποθήκης, το ενοίκιο λιανικής, τα έξοδα τρίτων και άλλα γενικά έξοδα - είναι εύκολο να καταλάβετε γιατί τα γυαλιά είναι πιο ακριβά.

Η αγορά online σας δίνει περισσότερη ελευθερία να αγοράσετε περισσότερα ζευγάρια στην ίδια τιμή, σαν εφεδρικά για να μην ξεμείνετε σε περίπτωση ατυχήματος.

7. Χρησιμοποιήστε προστατευτική θήκη & πανάκι μικροϊνών

Οι οργανικοί φακοί που χρησιμοποιούνται στα γυαλιά μπορεί να με σπάνε αλλά είναι πιο εύκολο να γρατσουνιστούν. Για το λόγο αυτό θα πρέπει να τα προσέχετε και να τα αποθηκεύετε σε μια σκληρή θήκη.

Όσο αφορά τον καθαρισμό των φακών, να χρησιμοποιείτε πανάκια με μικροίνες και όχι χαρτοπετσέτες ή τη μπλούζα σας. 

Συμβουλή: Να χρησιμοποιείτε και τα δύο χέρια για να τα βγάζετε τα γυαλιά σας (βοηθά στη διατήρηση του αρχικού τους σχήματος).
